# Password_Generator
A lightweight Python script that generates randomized passwords based on user-defined constraints. You can specify the total length and exactly how many numbers and symbols you want included.
Prerequisites

    Python 3.x installed on your machine.

    No external libraries required (uses standard string and random modules).
Installation

    Clone the repository:
    Bash
```
git clone https://github.com/Octahedron-apple/Password_Generator.git
```
Navigate to the directory:

```
cd Password_Generator
```
